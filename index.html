The Modern NBA MVP API

The NBA MVP API is a collection of MVP winners post the year 2000

API Documentation

Requests can be made with curl or other helper libraries by following regular REST calls. For example, here is how to GET the data for the year 2021:

https://basketball-api-100devs.herokuapp.com/api/2021

Calling this resource will respondf with the following object:

{
'name': 'Nikola Jokic',
'age': 26,
'ppg': 27.1,
'rpg': 13.8,
'apg': 7.9,
'status': 'Active'
}

Base URL

Users must prepend all resource calls with the base URL:

https://basketball-api-100devs.herokuapp.com/api/{year}
