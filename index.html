The Modern NBA MVP API

The NBA MVP API is a collection of MVP winners from 2000 onwards.

API Documentation

Requests can be made with curl or other helper libraries by following regular REST calls. For example, here is how to GET the data for the year 2021:

https://basketball-api-100devs.herokuapp.com/api/2021

Calling this resource will respond with the following object:

{
'name': 'Nikola Jokic',
'age': 26,
'ppg': 27.1,
'rpg': 13.8,
'apg': 7.9,
'status': 'Active'
}

API Endpoints

Users must prepend all resource calls with the base URL and include the parameter {year} for all API calls

https://basketball-api-100devs.herokuapp.com/api/{year}
